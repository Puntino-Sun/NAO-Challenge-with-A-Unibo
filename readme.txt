## Instructions
Starting from a working instance of the Docker container with Ubuntu 16.04 image, follow these steps to run the script:
  1. Install build-essential.

  2. Download and compile Python 2.7.3.

  3. Download and install the Python SDK for NAOqi.

  3. Download the source code of the project.

  4. Open the Choreography on the host machine and connect to the robot.

  5. Run the script inside a terminal :
```bash
PYTHONPATH=/pynaoqi/lib/python2.7/site-packages;QI_SDK_PREFIX=/pynaoqi /usr/local/bin/python NAO.py
```
